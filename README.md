# markdown-cv

A curriculum vitae maintained in plain text and rendered to HTML and PDF using CSS.

For more details, see the original [project page](http://elipapa.github.io/markdown-cv), or this blog post on [why one would want to switch to markdown for a CV](http://elipapa.github.io/blog/why-i-switched-to-markdown-for-my-cv.html).

I am grateful to Eliseo Papa ([Twitter](http://twitter.com/elipapa)/[Github](http://github.com/elipapa)/[Website](https://elipapa.github.io)) for designing this template and making it available under the [MIT License](https://github.com/elipapa/markdown-cv/blob/master/LICENSE).
